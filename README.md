# Deulababa - Server

The server side for the [Deulaubaba](https://github.com/niallantony/deulaubaba) mobile application. Written in Spring Boot this service provides users with the functionality to log and document learning progressions of their students.

## Features

- Designed to be deployed on Google Cloud Platform's App Service, with a PostgreSQL database and object storage.
- Comprehensive unit testing and integration testing ensures the service is reliable and robust
- A data model which reflects the domain, using JPA to implement the database layer
- OAuth security provides authentication and authorisation using JWT's from Auth0's third party service
